# RFT Automático - V10.2

Correção visual dos cards RFT.

- Remove as setas automáticas do `st.metric` nos cards RFT Diário/Semanal/Mensal/Anual/YTD.
- Aplica regra visual diretamente nos cards:
  - abaixo da meta = vermelho
  - acima ou igual à meta = verde
- Mantém V10.1: Pareto com Top configurável, estratificação por modelo, Tendência, Resumo executivo e Leitura diária do RFT.
