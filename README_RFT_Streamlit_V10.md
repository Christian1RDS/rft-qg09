# RFT Automático - V10

Versão com retorno dos recursos do Dashboard e da aba Tendência.

Inclui:
- Aba Tendência com gráfico mensal e semanal em barras/colunas.
- Dashboard com Meta x resultado.
- Dashboard com Resumo executivo do recorte.
- Dashboard com Leitura diária do RFT em linha.
- Pareto clássico em SVG/HTML interno, sem Plotly e sem Matplotlib.
- Leitura robusta de CSV tabulado/UTF-16.

Deploy:
```text
app_rft_streamlit_v10.py
requirements.txt
```

Main file path:
```text
app_rft_streamlit_v10.py
```
